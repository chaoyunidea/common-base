## 弹窗 {docsify-ignore-all}
 
**标签名：fe-popup**

```
  <template>
      <fe-popup size="small" hideClose="" isCenter=""></fe-popup>
  </template>

```


 
## 标签属性说明

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 弹窗大小 | String |  'normal'  |
| hideClose | 是否有关闭按钮 | Bool |  true  |
| isCenter | 是否居中 | Bool | false |
