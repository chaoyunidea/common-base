# 移动端公共组件
 
 - 弹窗通用组件: 可通过挂载在全局对象app下面的方法调用显示出的功能组件
     
 - 标签组件: Vue已全局挂载的标签组件，无需使用compontents声明便可使用
   
 - 公共方法: 功能方法，筛选过滤等
 
 - 公共样式方法


## 常用部分
  **以下常用部分必须掌握学习**

  - 弹窗组件: 日期选择，提示
  - 标签组件: 弹窗，loading加载,上下滑动组件,时间区间
  - 公共方法： 
    - 过滤器 时间格式处理
    - 混入  弹窗基本事件与操作混入
    - 方法 clone，filterDate，setDocTitle
