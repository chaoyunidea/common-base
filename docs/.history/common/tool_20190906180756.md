``克隆一个新的对象或数组，不能克隆重复引用本身对象``

``为了在移动端调试方便,因为在.vue的template里面无法直接调用 window.alert``

`` 将类似location中的字符串查询参数解析为对象   例如http://www.baidu.com?name=jiajie&age=31&sex=1 ``

`` 将location中的hash解析为包含hash、查询参数的对象 ``

``将简单json对象转化为查询参数字符串 ``
