##  默认吐司  

*  API
  * ` app.toast(content) `
  * ` app.toast( type, text) `
  * ` app.toast( text, duration)   //text不为参数type中的值时 为此效果 `
  * ` app.toast( type, text, duration) `
  * ` app.toast( {type, text, duration}) `

参数说明:

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| content | 提示内容 | String | |
| type | 提示类 |  success  ,  error  ,  info  ,  default  | default |
| duration | 持续时间 | Number | 2000 |

* 示例：
  
```
   app.toast('自爆装置已启动', 3000 )

```
