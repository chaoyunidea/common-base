## 普通滚动元素

### 示例 {docsify-ignore}

```
<template>
  <scroller-base class="scroller" :data="contentHtml">
    ...content
  </scroller-base>
</template>

<script>
   export default {
     data(){
       return {
         contentHtml:{} //会监控此数据的变化来动态刷新滚动条
       }
     }

   }
</script>
```

#### 属性说明 {docsify-ignore}

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| sdate | 开始日期 | String |    |
